私たち情報工学科は八王子国際キャンパスで勉強に日々励んでおり、学部長の高橋丈博教授を中心として多くの先生方と研究をしています。:+1:  

#####1.情報工学科コース
拓殖大学工学部情報工学科には三つのコースが存在し
それぞれ
* システムエンジニアリングコース
* インターネット＆セキュリティコース
* 知能情報コース　　　　　　　　　　　　があります。
#####2.情報工学科　研究室う
また、情報工学科にはそれぞれ12個の研究室がありそれぞれ違った教員が担当しています。
